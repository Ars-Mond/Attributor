Настольное приложение для редактирования метаданных фотографий (EXIF/XMP) перед отправкой в фотобанки. Заполняйте заголовок, описание, ключевые слова и категории вручную или генерируйте их локальной AI vision-моделью, а затем записывайте их в файл изображения без перекодирования. Правки хранятся в локальной базе данных по мере работы и записываются в файл при сохранении.

**Поддерживаемые форматы:** JPEG · PNG · WebP

## Возможности

- **Редактирование метаданных** — заголовок, описание, ключевые слова, категории и имя файла релиза, записываемые в изображение как XMP/EXIF/IPTC без перекодирования
- **AI-автоатрибутация** — генерация заголовка, описания, ключевых слов, категорий и флагов контента (редакционное / 18+ / иллюстрация) локальной vision-моделью Ollama, для одного фото или целого пакета
- **Локальное хранилище метаданных** — правки и результаты AI хранятся в базе данных приложения по мере работы (статус «в приложении») и записываются в файл по кнопке **Сохранить**; **Сброс** возвращает к файлу, а внешние изменения файла обнаруживаются с запросом «оставить версию приложения или файла»
- **Экспорт в CSV** — экспорт сохранённых метаданных в таблицы CSV, по одному файлу на фотосток, в выбранную папку; настраиваемые пресеты для каждого стока (столбцы, типы значений, разделитель) с готовыми пресетами для Shutterstock, iStock, Adobe Stock и Envato
- **Многоязычный интерфейс** — английский и русский, переключается в настройках и определяется по языку ОС при первом запуске
- **Пакетный режим** — выбор нескольких файлов и одновременное редактирование общих метаданных; для ключевых слов видно, в каких файлах они присутствуют, с управлением распространения/удаления по каждому слову
- **Автодополнение ключевых слов** — нечёткий поиск по более чем 1 000 стоковых терминов с изменением порядка перетаскиванием, копированием/вставкой из буфера обмена и массовым вводом через запятую
- **Автосохранение** — опциональное автосохранение с задержкой 1 с для редактирования одного файла
- **Наблюдение за файлами** — обнаруживает внешние изменения (переименования, удаления) и автоматически обновляет дерево файлов
- **Гибкая раскладка панелей** — три панели с изменяемым размером, закрываемые и переставляемые (Управление · Просмотр · Иерархия)
- **Несколько режимов отображения** — таблица, содержимое и значки с вертикальной/горизонтальной раскладкой для браузера файлов
- **Переименование при сохранении** — изменение поля имени файла переименовывает сам файл на диске

## Известные ограничения

- Режим «Значки» показывает только файлы верхнего уровня открытой папки (без обхода подпапок)
- Форматы TIFF и RAW не поддерживаются
- Автосохранение отключено в пакетном режиме во избежание случайной перезаписи

## Для разработчиков

**Стек:** Rust · Tauri 2 · SvelteKit · TypeScript · Vite · SCSS

**Требования:** [Rust](https://rustup.rs) · [Node.js](https://nodejs.org) · [pnpm](https://pnpm.io) · [Требования Tauri](https://tauri.app/start/prerequisites/)

```sh
# Установка зависимостей фронтенда
pnpm install

# Запуск в режиме разработки
cargo tauri dev

# Сборка релиза
cargo tauri build

# Перегенерация иконки приложения из исходного изображения
cargo tauri icon src-tauri/icons/logo.png
```
