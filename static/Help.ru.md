# Attributor — Руководство пользователя

Attributor — это настольное приложение для редактирования **XMP/EXIF-метаданных** стоковых фотографий перед их отправкой в агентства, такие как Shutterstock, Adobe Stock, Getty Images и другие. Оно позволяет встраивать заголовок, описание, ключевые слова и категории прямо в файлы изображений без их перекодирования, сохраняя полное качество изображения.

**Поддерживаемые форматы:** JPEG · PNG · WebP

---

## Начало работы

### Открытие папки

Перейдите в **Файл → Открыть папку…** или нажмите значок папки на панели «Иерархия». Attributor сканирует выбранную папку и отображает все поддерживаемые файлы изображений в дереве файлов.

Последняя открытая папка запоминается и автоматически восстанавливается при следующем запуске.

### Выбор файла

Щёлкните любой файл на панели «Иерархия», чтобы открыть его. Изображение появляется на панели **Просмотр**, а его существующие метаданные загружаются в панель **Управление**.

Если в файле есть несохранённые изменения, Attributor предложит **Сохранить**, **Не сохранять** или **Отмена** перед переключением на другой файл.

---

## Редактирование метаданных

Все поля метаданных находятся на панели **Управление**.

| Поле | Описание |
|-------|-------------|
| **Имя файла** | Основа имени файла (без расширения). Переименование происходит при сохранении. |
| **Заголовок** | Короткий описательный заголовок изображения (хранится как `dc:title`). |
| **Описание** | Подробное описание для поисковых систем и покупателей (`dc:description`). |
| **Ключевые слова** | Теги, описывающие содержимое изображения (`dc:subject`). |
| **Категории** | Необязательные категории содержимого (`photoshop:Category`). |
| **Файл релиза** | Имя документа релиза модели/собственности, если применимо. |

**Счётчик слов / символов** рядом с полями «Заголовок» и «Описание» помогает не выходить за пределы лимитов агентств.

### Сохранение

Нажмите **Сохранить изменения** или включите **Автосохранение**, чтобы записывать изменения в файл изображения автоматически после 1 секунды бездействия. Индикатор состояния в заголовке панели показывает:

- 🟢 **просмотр** — файл загружен, изменений нет
- 🟡 **правка** — есть несохранённые изменения
- 🔵 **пакет** — выбрано несколько файлов

## Ключевые слова

### Добавление ключевых слов

- Введите ключевое слово в поле ввода и нажмите **Enter**.
- Введите несколько ключевых слов, разделённых **`, `** (запятая-пробел), чтобы добавить их все сразу.
- Выберите вариант из **выпадающего списка автодополнения**, который появляется по мере ввода (нечёткий поиск по более чем 1 000 стоковых терминов).
- Щёлкните любой пресет **Стоковые ключевые слова**, чтобы добавить предопределённые теги по категориям.

### Управление ключевыми словами

- **Перетаскивайте** чипы, чтобы изменить их порядок.
- Щёлкните **×** на чипе, чтобы удалить его.
- **Копировать** — копирует все ключевые слова в буфер обмена в виде списка, разделённого запятыми.
- **Вставить** — разбирает текст из буфера обмена и добавляет все найденные ключевые слова.

## Пакетный режим

Выберите **два или более файла** на панели «Иерархия» (с помощью `Shift`+щелчок или `Ctrl`+щелчок), чтобы войти в пакетный режим. Панель «Управление» переключается в режим редактирования нескольких файлов.

### Как работает пакетное редактирование

- **Заголовок / Описание / Категории** — у каждого поля есть флажок. Установите его, чтобы применить значение ко всем выбранным файлам при сохранении.
- Если у файлов уже есть разные значения для поля, в подсказке отображается **(разные значения)**, и флажок по умолчанию снят.
- **Ключевые слова** — показывается объединённый набор ключевых слов всех файлов:
  - **Сплошной чип** — ключевое слово присутствует во **всех** выбранных файлах.
  - **Пунктирный серый чип** — ключевое слово присутствует только в **некоторых** файлах. Перед ним появляется закрашенный кружок ● — щёлкните по кружку, чтобы распространить ключевое слово на все файлы.
- Щёлкните **×** на любом чипе, чтобы удалить это ключевое слово из всех файлов при сохранении.
- Перетаскивайте чипы, чтобы изменить их порядок.
- **Копировать** — копирует только общие (сплошные) ключевые слова.
- **Вставить** — добавляет ключевые слова во все выбранные файлы.

### Предпросмотр в пакетном режиме

На панели «Просмотр» отображается **последний выбранный файл**. Чтобы просмотреть конкретный файл из вашего выбора, не изменяя его, удерживайте **Alt** и щёлкните файл на панели «Иерархия».

### Сохранение пакетных изменений

Нажмите **Сохранить N файлов**, чтобы применить все изменения. Каждый файл обновляется по отдельности: перезаписываются только отмеченные поля; неотмеченные поля сохраняют свои исходные значения.

## Система докинга

В Attributor есть три панели, которые можно свободно перемещать:

| Панель | Положение по умолчанию | Назначение |
|-------|-----------------|---------|
| **Управление** | Слева | Форма редактирования метаданных |
| **Просмотр** | По центру | Предпросмотр изображения |
| **Иерархия** | Справа | Дерево файлов / браузер |

### Изменение размера панелей

Перетащите разделитель между любыми двумя панелями, чтобы изменить их размер. Раскладка сохраняется автоматически.

### Закрытие и восстановление панелей

Щёлкните **×** в строке заголовка панели, чтобы закрыть её. Восстановите её через **Окна → Показать панель управления / Показать иерархию** в строке меню. Восстановленные панели возвращаются на своё последнее место.

### Перестановка панелей

Перетащите строку заголовка панели и отпустите её на **зоне сброса** (подсвеченные края) другой панели, чтобы разделить пространство и разместить её в любом месте раскладки.

## Режимы отображения панели «Иерархия»

Используйте панель инструментов в панели «Иерархия», чтобы переключать способ отображения файлов.

### Режимы отображения

| Значок | Режим | Описание |
|------|------|-------------|
| ☰ | **Таблица** | Компактный список с именами файлов и деревом папок. |
| ▤ | **Содержимое** | Строка с миниатюрой и именем файла. |
| ⊞ | **Значки** | Сетка крупных миниатюр. |

### Направление раскладки (режимы «Содержимое» и «Значки»)

| Значок | Направление | Поведение |
|------|-----------|-----------|
| — | **Вертикально** | Элементы располагаются сверху вниз; каждое изображение занимает всю ширину панели. |
| ǀ | **Горизонтально** | Элементы выстраиваются слева направо в одну прокручиваемую строку; каждое изображение занимает всю высоту панели. Прокручивайте колёсиком мыши. |

## Сочетания клавиш

### Навигация по файлам

| Клавиша | Действие |
|-----|--------|
| `↑` / `↓` | Переместить выделение вверх / вниз в списке файлов |
| `Shift` + `↑` / `↓` | Расширить диапазон выделения |
| `Shift` + щелчок | Выбрать диапазон файлов |
| `Ctrl` + щелчок | Добавить / убрать отдельный файл из выделения |
| `Alt` + щелчок | Просмотреть выбранный файл на панели «Просмотр» (только в пакетном режиме, выделение не меняется) |

### Редактирование метаданных

| Клавиша / Ввод | Действие |
|-------------|--------|
| `Enter` | Добавить введённое ключевое слово |
| `, ` (запятая-пробел) | Добавить предшествующий текст как ключевое слово и продолжить ввод |
| `Esc` | Закрыть подсказки автодополнения / закрыть диалоги |
| `↑` / `↓` в подсказках | Перемещаться по списку автодополнения |

### Контекстное меню (текстовые поля)

Щёлкните правой кнопкой мыши по любому полю ввода или текстовой области, чтобы открыть контекстное меню:

| Пункт | Сочетание | Действие |
|------|----------|--------|
| **Копировать** | `Ctrl+C` | Скопировать выделенный текст в буфер обмена |
| **Вставить** | `Ctrl+V` | Вставить текст из буфера обмена в позицию курсора |

## Советы

- **Переименование при сохранении** — изменение поля «Имя файла» переименовывает сам файл при сохранении. Панель и дерево файлов обновляются автоматически.
- **Наблюдение за файлами** — если файлы перемещаются или удаляются извне, пока Attributor открыт, дерево обновляется автоматически, и появляется уведомление, если текущий открытый файл исчез.
- **Автосохранение + пакет** — в пакетном режиме автосохранение отключено, чтобы предотвратить случайную перезапись. Используйте кнопку **Сохранить N файлов** явно.
